Cloud Lines is an innovative new product from MA Systems Ltd, which enables our valued customers to keep accurate fully-featured records of their breeding programmes.

Cloud Lines was developed in 2018 due to a growing need to support customers using bespoke or legacy database applications. Cloud Lines is built on the most modern IT architecture, and aims to deliver superior service at reduced cost. The cloud-based platform which hosts Cloud Lines can boast of 99.9% uptime and 99.9999% data durability. All user data is regularly backup up as standard and customers have the option to export data as often as needed.

Cloud Lines is designed to be both user friendly and as fully featured as possible. All Levels of customer have access to the same breed-specific functionality, including automatically generated pedigree certificates which can be exported as PDFs.
